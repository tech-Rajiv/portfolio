Project 1: BILL.EZ

✨ Bill.Ez (Sales management app)– Solving Real Problems✨

Live Visit
Github Link

During Diwali, I ran a 10-day firecracker shop, and it came with a host of challenges. This project became my first and most crucial solution, pushing me to solve a real-life problem with technology.

- The Challenge

  • Staff struggled to memorize prices.
  • Customers doubted verbal pricing and often bargained aggressively.
  • Competitors nearby (within 200 meters) made it harder to stand out.
  • i only had 20days time left for diwali , so i needed to comeup with a solution as early as possible!.

️- The Solution

I developed a Sales Management Web App for my stafs with a focus on:

  • Core Functionality – Streamlining product management, sales, and payment workflows.
  • Mobile-Friendly UI – Optimized specifically for mobile devices.
  • Functionality over asthetics – as i had a limited time so i decide to stick to basic javascript rather than exploring fancy frameworks.

Key Features

  • Added Products with Categories and images & prices
  • Manage & Generate Totals
  • Search, Sort, and Save Sales Data
  • QR Code Generation for Payments
  • Saving sales in Database so that i can get overview
  • Export Sales Reports as PDFs

🚀 The Impact

  • Customer Trust – Digital pricing eliminated doubts and built transparency.
  • Competitive Edge – The professional app interface stood out among nearby shops.
  • Reduced Bargaining – Customers were satisfied with the final total displayed.
  • Seamless Operations – My staff served multiple customers efficiently.

This project saved time, boosted efficiency, and gave my shop a professional edge. View this project on a mobile device for the best experience!
